Willkommen in meinem IT-Infrastruktur-Portfolio! Dieses Projekt demonstriert den Aufbau eines vollständigen, simulierten Firmennetzwerks von der Bare-Metal-Virtualisierung bis hin zum zentralen Identitätsmanagement.

# Projektziel
Aufbau einer isolierten, sicheren und skalierbaren IT-Infrastruktur auf Basis von Proxmox VE, um praxisnahe Administration, Netzwerksegmentierung und Sicherheitskonzepte zu demonstrieren.

# Eingesetzte Technologien
- **Hypervisor:** Proxmox VE (Bare-Metal, KVM, LVM/ZFS)
- **Networking & Security:** pfSense (VLAN-Routing, NAT, Firewalling)
- **Identity & Access Management:** Windows Server 2022 (Active Directory, DNS, DHCP, GPOs)
- **File Services:** Windows Server SMB-Shares mit NTFS-Berechtigungen & Quotas
- **Containerization & Monitoring:** Ubuntu Server, Docker, Zabbix
- **Backup & Disaster Recovery:** Proxmox Backup-Konzepte (Live-Snapshots)
